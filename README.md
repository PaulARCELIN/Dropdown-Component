### Install

Install using npm i dropdown-paul-arcelin

### Usage

In a React app, use the dropdown component:
import { Dropdown } from 'dropdown-paul-arcelin'

#### Dropdown props
List Array
Label 	String
id 	String
selection function
| Name        | Description      
| ----------- | -----------      
| List        | Array            
| Label       | String             
| id          | String             
| selection   | Function
